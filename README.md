Book Price Web Scraper (Python)

Overview

A basic Python web scraping project that extracts book titles and prices (in INR) from an online bookstore and saves the data into a CSV file. This project demonstrates foundational web scraping and data handling skills.

Tech Stack
	•	Python
	•	Requests
	•	BeautifulSoup
	•	CSV / Pandas

Files
	•	scraper.py – scraping logic
	•	books_data_inr.csv – extracted data
	•	README.md – documentation

How to Run
pip install requests beautifulsoup4 pandas
python scraper.py
Output

Generates a CSV file containing book titles and prices in INR.

Purpose

Learning and practicing Python web scraping fundamentals.
